# Understanding America v0.5.0 - Publication Milestone

This release marks the first publication milestone for **Understanding America**, an interactive civic learning handbook covering American history, government, the Constitution, citizenship, and USCIS preparation.

## Included

- Interactive HTML book
- Chapter navigation
- Government Explorer
- Constitution Explorer
- Timeline
- Visual Gallery
- Flashcards
- Quiz starter
- Glossary starter
- Print preparation pages
- Publication QA materials
- Editorial QA and fact-checking workflow
- Accessibility audit starter
- Source register

## Downloadable Packages

- `Understanding-America-site.zip`  
  Full HTML website package.

- `Understanding-America-print.zip`  
  Print/PDF preparation package.
