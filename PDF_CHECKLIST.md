# PDF Production Checklist

## Preflight
- [ ] Confirm all chapters are linked from `index.html`
- [ ] Confirm print stylesheet loads correctly
- [ ] Confirm no broken internal links
- [ ] Confirm images and SVGs print correctly
- [ ] Confirm tables do not overflow page width
- [ ] Confirm chapter pages start on new pages
- [ ] Confirm cover and title page print cleanly

## Content
- [ ] Final proofreading complete
- [ ] USCIS facts verified
- [ ] Current officials verified
- [ ] Source notes complete
- [ ] Glossary complete
- [ ] Index complete

## Print Layout
- [ ] Page size: Letter
- [ ] Margins verified
- [ ] Font sizes readable
- [ ] Page breaks reviewed
- [ ] Headings not orphaned
- [ ] Callout boxes not split awkwardly
- [ ] Long tables reviewed

## Export
- [ ] Browser print tested
- [ ] Playwright/Puppeteer PDF tested
- [ ] Final PDF opened locally
- [ ] Final PDF page count recorded
- [ ] Final PDF uploaded to release
